# Project Abyss

ARPG Top-Down indie inspirado en Diablo, desarrollado en Unity 6 LTS con C#.

## Assets de Terceros

Los assets externos no están incluidos en el repositorio por restricciones de licencia.
Para correr el proyecto localmente, descargá los siguientes packs y copiados en la carpeta `Assets/ThirdParty/`:

### Personaje y Enemigos
- **Tiny RPG Character Asset Pack v1.03** por Zerie
- Descarga: https://zerie.itch.io/tiny-rpg-character-asset-pack
- Carpeta destino: `Assets/ThirdParty/Tiny RPG Character Asset Pack v1.03 -Free Soldier&Orc/`

## Stack Tecnológico
- Unity 6 LTS (Universal 2D - URP)
- C#
- Git + GitHub
